# Contributing to pr-template-validator

All contributions are welcomed (of any kind)!

## Features and bug reports

Feel free to open up an issue that describes a requested feature or a bug report

## Pull Request

If you'd like to contirubte code that's awesome! Feel free to fork & checkout this repository and send a PR. Remember to
make all of the tests pass and try to cover your changes with tests. You don't have to aim for 100% test coverage.
